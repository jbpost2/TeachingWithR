TeachingWithR Course Files
================

<p style="font-family: Lucida Console;">

This section of the course introduces you to programming in the SAS
software. SAS is an extremely powerful language that can fit almost any
kind of model, is backward compatible (code you write now will work in
10 years), and has great user support.

</p>

<p style="font-family: Lucida Console;">

We’ll utilize and learn about the SAS Studio environment. We’ll learn
how to read in common types of raw data including delimited data, excel
data, JSON data, and more\!

</p>

<p style="font-family: Lucida Console;">

Estimated time for completion: 4-6 hours 

</p>
